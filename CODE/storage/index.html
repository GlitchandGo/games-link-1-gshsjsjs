How to switch from HTTPS to HTTP:
  1. Click the address bar at the top of your browser.
  2. Check if the URL starts with **https://**.
  3. Manually change **https://** to **http://**.
  4. Press **Enter** to reload the page using HTTP.
If your browser **forces HTTPS automatically**, try using **Incognito mode** or a different browser.
